\# 企业\_报表模板提供器（Enterprise Financial Report Template Provider）



\*\*版本\*\*：1.0（基于财政部财会〔2019〕6号一般企业财务报表格式）



\*\*触发关键词\*\*：企业报表 / 企业会计准则模板 / 一般企业资产负债表 / 利润表 / 现金流量表



\*\*功能描述\*\*：

\- 自动返回企业会计准则四大标准报表空模板（资产负债表、利润表、现金流量表、所有者权益变动表）

\- 支持 JSON 输出（便于报表模块直接渲染/填充数据）

\- 严格按照新准则列报要求（含交易性金融资产、合同资产/负债、使用权资产、研发费用单独列示等）

\- 支持数据填充后自动计算小计、合计、净利润、净增加额

\- 附带比较信息（期末/上年年末、本期/上期）处理逻辑



\*\*输入参数\*\*（您的软件可传入）：

\- 编制单位、会计期间（年/月）

\- 数据源（可选：JSON 账簿数据）

\- 输出格式（Markdown / JSON / Excel 模板）



\*\*输出\*\*：四大报表 + 附注提示
