# NGO_报表模板提供器（NGO Financial Report Template Provider）

**版本**：1.0（基于财政部《民间非营利组织会计制度》最新格式）  
**触发关键词**：NGO报表 / 民间非营利组织模板 / 业务活动表 / NGO资产负债表  

**功能描述**：
- 自动返回 NGO 准则三大标准报表空模板（含行次、项目名称、年初/期末、本月/本年累计、限定性/非限定性列）
- 支持 JSON 输出（便于报表模块直接渲染/填充数据）
- 自动区分「限定性净资产」与「非限定性净资产」（NGO 核心要求）
- 附带受托代理资产/负债处理逻辑
- 支持数据填充后自动计算合计与净资产变动额

**输入参数**（您的软件可传入）：
- 编制单位、会计期间（年/月）
- 数据源（可选：JSON 账簿数据）
- 输出格式（Markdown / JSON / Excel 模板）

**输出**：三大报表 + 附注提示
