# 输出规则

## 1. 输入约定

推荐输入字段：

- `entity_name`: 编制单位名称
- `period_label`: 用户希望显示的期间文字
- `month`: 资产负债表月末期间，格式 `YYYY-MM`
- `start_period` / `end_period`: 动态报表期间，格式 `YYYY-MM`
- `output_format`: `markdown` | `json` | `excel-friendly`
- `blank_template`: 是否仅输出空模板
- `include_note_prompts`: 是否附带附注提示
- `amount_unit`: 默认 `元`

## 2. 期间口径

- 资产负债表：
  默认按月末编报，显示 `年 月 日` 截至时点。
- 利润表：
  默认按期间编报，显示 `年 月`。
- 现金流量表：
  默认按期间编报，显示 `年 月`；若用户明确要年度模板，可显示 `年度`。
- 所有者权益变动表：
  默认按年度模板输出；若用户明确要求中期模板，只能在保留官方矩阵结构的前提下替换表头期间，不能擅自删减矩阵列。

## 3. 空模板规则

- 空模板数值统一留空，不写死 `0` 或 `0.00`。
- 保留全部官方行次、总计行和层级。
- 若用户要求“按重要性删减”，可以删减不存在相应业务的报表项目，但必须附一句说明：
  `已按企业实际不存在业务项目做必要删减，其余列报口径保持企业会计准则要求。`

## 4. JSON 规则

- 使用 `report-templates.json` 中的字段名和顺序。
- `formula` 仅表达计算关系，不代表已自动算值。
- 空模板金额字段使用 `null`。
- 已填值模板金额字段使用数字，不加千分位和货币符号。
- `matrix` 结构仅用于所有者权益变动表。

## 5. Excel 友好规则

- 每张报表单独一个 sheet。
- 建议 sheet 名：
  - `资产负债表`
  - `利润表`
  - `现金流量表`
  - `所有者权益变动表`
- 每个 sheet 顶部保留：
  - 报表名称
  - 编制单位
  - 期间/日期
  - 单位

## 6. 校验链

### 资产负债表

- `流动资产合计 = 各流动资产项目之和`
- `非流动资产合计 = 各非流动资产项目之和`
- `资产总计 = 流动资产合计 + 非流动资产合计`
- `流动负债合计 = 各流动负债项目之和`
- `非流动负债合计 = 各非流动负债项目之和`
- `负债合计 = 流动负债合计 + 非流动负债合计`
- `所有者权益合计 = 各权益项目之和`
- `资产总计 = 负债和所有者权益总计`

### 利润表

- `营业利润 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 财务费用 + 其他收益 + 投资收益 + 以摊余成本计量的金融资产终止确认收益 + 净敞口套期收益 + 公允价值变动收益 + 信用减值损失 + 资产减值损失 + 资产处置收益`
- `利润总额 = 营业利润 + 营业外收入 - 营业外支出`
- `净利润 = 利润总额 - 所得税费用`
- `综合收益总额 = 净利润 + 其他综合收益的税后净额`

### 现金流量表

- `经营活动现金流量净额 = 经营活动现金流入小计 - 经营活动现金流出小计`
- `投资活动现金流量净额 = 投资活动现金流入小计 - 投资活动现金流出小计`
- `筹资活动现金流量净额 = 筹资活动现金流入小计 - 筹资活动现金流出小计`
- `现金及现金等价物净增加额 = 三类净额之和 + 汇率变动影响`
- `期末现金及现金等价物余额 = 期初现金及现金等价物余额 + 现金及现金等价物净增加额`

### 所有者权益变动表

- `本年年初余额 = 上年年末余额 + 会计政策变更 + 前期差错更正 + 其他`
- `本年年末余额 = 本年年初余额 + 本年增减变动金额`
- 上年金额区块与本年金额区块使用同一列结构。

## 7. 附注提示

建议默认附带简短提示，不展开长篇附注正文。优先提示：

- 货币资金受限情况
- 应收款项及坏账准备政策
- 合同资产/合同负债确认政策
- 使用权资产与租赁负债
- 研发费用费用化/资本化政策
- 公允价值变动与减值
- 现金及现金等价物口径
- 所有者权益变动中的政策变更、差错更正和留存收益结转

## 8. 错误修正规则

- 若用户给出的模板把 `合同负债` 放到资产端，必须改正。
- 若用户遗漏 `使用权资产`、`租赁负债` 等现行模板项目，必须补齐。
- 若用户把所有者权益变动表改成单栏简化版，必须恢复为官方矩阵结构。
