





\# Dude Accounting (单机版记账软件) - 产品需求文档 (PRD) v2.0



\## 1. 产品概述与技术架构

\*   \*\*产品定位：\*\* 面向企业日常记账的跨平台单机版财务软件。

\*   \*\*运行环境：\*\* Windows / macOS 桌面端。

\*   \*\*推荐技术栈（面向 AI 编程工具指定）：\*\* 

&nbsp;   \*   前端框架：React 或 Vue3 (结合 TypeScript 保证财务数据类型严谨性)。

&nbsp;   \*   桌面端框架：Electron 或 Tauri。

&nbsp;   \*   本地数据库：SQLite (轻量、单机、支持复杂查询)。

\*   \*\*核心特点：\*\* 多账套管理、支持跨年查询、基于快捷键的高效凭证录入、动态财务报表、多端网盘备份。



\## 2. 全局工作流与基础规则

\*   \*\*标准业务流：\*\* 登录 -> 选择/新建单位账套 -> 录入记账凭证 -> 审核凭证 -> 记账 -> 期末损益结转 -> 结账。

\*   \*\*初始化：\*\* 系统内置超级管理员账号 `admin`，初始无密码。

\*   \*\*无密码登录规则：\*\* 支持账号密码为空的情况，若账号未设置密码，输入账号后点击登录/回车即可进入系统。



\## 3. UI/UX 架构设计 (基于原型图)

\*   \*\*登录界面：\*\* 极简风格，包含软件Logo/名称，账号输入框，密码输入框，登录按钮。

\*   \*\*主功能界面：\*\*

&nbsp;   \*   \*\*左侧导航栏：\*\* 垂直排列的主功能模块按钮（账套设置、账务处理、账簿查询、报表输出、系统设置）。

&nbsp;   \*   \*\*顶部信息栏：\*\* 右侧显示当前“账套名称”和“会计期间”。点击可唤出下拉列表进行切换。

&nbsp;   \*   \*\*工作标签区 (Tab Bar)：\*\* 位于顶部信息栏下方，显示当前打开的子功能页面。支持多标签页切换，带关闭(x)按钮。

&nbsp;   \*   \*\*作业区 (Workspace)：\*\* 占据右侧主要区域，显示当前标签页的具体功能。

\*   \*\*全局悬浮交互逻辑 (核心要点)：\*\*

&nbsp;   \*   在任何已打开的子功能 Tab 页面下，点击左侧主功能按钮，将\*\*模糊处理当前作业区\*\*，并在作业区上方\*\*悬浮显示\*\*该主模块对应的子功能按钮组。

&nbsp;   \*   \*点击悬浮区域外：\* 退出悬浮状态，取消模糊，恢复当前工作区操作。

&nbsp;   \*   \*点击悬浮的子功能按钮：\* 新建对应的 Tab 标签并打开，退出悬浮状态。



\## 4. 核心功能模块详细说明



\### 4.1 账套设置 

\*   \*\*会计科目设置：\*\* 左侧树形结构（默认全收起），支持通过独立 Dialog 新增、修改科目。

\*   \*\*辅助账设置：\*\* 支持辅助核算项目的新增、修改（独立 Dialog）。

\*   \*\*期初数录入：\*\* 录入各科目期初余额，底部实时显示试算平衡状态（借方合计=贷方合计校验）。

\*   \*\*现金流量匹配设置：\*\* 建立“会计科目”与“现金流量项目”的映射规则，提供初始默认规则。

\*   \*\*期末损益结转设置：\*\* 设置收入、费用类科目结转至“本年利润”等科目的对应关系规则。提供默认规则。



\### 4.2 账务处理 

\*   \*\*凭证录入：\*\*

&nbsp;   \*   \*\*表格输入区：\*\* 摘要、会计科目、借方金额、贷方金额。

&nbsp;   \*   \*\*快捷键与自动化（高优实现）：\*\*

&nbsp;       \*   输入科目代码，联想下拉带出科目名称。

&nbsp;       \*   金额输入框按 `=` 键，自动计算并填入借贷平衡金额。

&nbsp;       \*   按 `Enter` 键光标自动向右移动；最后一行按 `Enter` 自动新增一行。

&nbsp;       \*   保存时，自动校验并清理无内容的空行。

&nbsp;   \*   \*\*现金流量强制拦截：\*\* 录入涉及现金流科目时，必须匹配现金流量项目（未匹配则阻断保存/下一张操作），强制要求点击右侧“现金流编辑按钮”指定。

\*   \*\*期末损益结转：\*\*

&nbsp;   \*   一键触发，自动结转本月损益类账户并生成结转凭证。

&nbsp;   \*   \*\*防呆合规逻辑：\*\* 依据系统设置中的【允许制单人与审核人为同一人】开关决定凭证状态。若开启，凭证自动为已审核已记账（审核/记账人为当前账号）；若关闭，则生成处于“未审核”状态的凭证，需切换账号完成后续审批。

\*   \*\*凭证管理：\*\*

&nbsp;   \*   列表显示，默认查询本月，支持跨年。

&nbsp;   \*   支持勾选记录进行：审核、记账、反记账、反审核、删除、打印。

&nbsp;   \*   \*\*打印引擎：\*\* 自适应 A4 纸张，默认一页纸上下打印两张凭证，支持设置两张凭证间的垂直间距。

\*   \*\*结账：\*\*

&nbsp;   \*   \*\*前置硬性拦截规则 (必须全部满足)：\*\*

&nbsp;       1. 上一个月已结账（首月除外）。

&nbsp;       2. 本月所有有效凭证已记账。

&nbsp;       3. 本月所有损益类科目余额已结转清零。

\*   \*\*凭证查询：\*\*

&nbsp;   \*   支持日期范围、金额范围、摘要关键词、辅助核算项的单选/多选组合查询。

&nbsp;   \*   \*\*金额范围逻辑：\*\* 查询条件定义为“单张凭证的\*\*借方\*\*或\*\*贷方\*\*发生额包含在设定的金额范围内”。



\### 4.3 账簿查询 

本模块内\*\*所有\*\*账簿均支持跨年度查询、自定义日期范围查询、\*\*数据导出（Excel 或 CSV 格式）\*\*以及打印。

\*   \*\*总账余额表 / 科目余额表 / 辅助余额表：\*\*

&nbsp;   \*   \*\*下钻交互：\*\* 右键点击某行条目，弹出上下文菜单（Context Menu）。选择“查询明细账”后，新建 Tab 标签打开对应明细账，并自动带入该条目的查询参数。

\*   \*\*科目明细账 / 序时账 / 辅助明细账：\*\*

&nbsp;   \*   \*\*溯源交互：\*\* 右键点击具体明细/流水条目，弹出上下文菜单。选择“查询凭证”后，新建 Tab 或弹窗展示对应的原始凭证内容。



\### 4.4 报表输出 

\*   \*\*动态渲染逻辑：\*\* 视当前账套引用的“会计准则”动态展示报表菜单及模板。

&nbsp;   \*   \*企业会计准则：\* 资产负债表、利润表、现金流量表。

&nbsp;   \*   \*民办非企业准则：\* 资产负债表、业务活动表、现金流量表。



\### 4.5 系统设置 

\*   \*\*系统参数设置：\*\* 

&nbsp;   \*   提供【允许制单人与审核人为同一人】全局开关。

\*   \*\*账号管理 (仅 admin 操作)：\*\*

&nbsp;   \*   管理登录名、真实姓名、密码。

&nbsp;   \*   权限分配 (勾选)：凭证录入、审核、记账、系统设置权限、账套设置权限。`admin` 锁定拥有所有权限且不可删除。

\*   \*\*会计准则设置：\*\*

&nbsp;   \*   提供会计准则（如企业会计准则、民非准则）的科目体系和报表模板的\*\*手动新增、修改、维护\*\*功能。

&nbsp;   \*   \*注（给开发者的指令）：无需开发任何外部文件（如 SKILL 文件）的解析逻辑。初始默认的标准准则数据，请在 Vibe Coding 开发阶段通过初始化脚本/Seed 数据直接写入数据库。\*

\*   \*\*账套备份：\*\*

&nbsp;   \*   \*\*本地备份：\*\* 选择本地路径生成数据库备份文件。

&nbsp;   \*   \*\*网盘备份：\*\* 支持 Google Drive, 百度网盘, OneDrive。

&nbsp;   \*   \*\*备份触发与异常阻断逻辑：\*\* 

&nbsp;       \*   支持勾选“月末结账后自动备份”或手动点击“立即备份”。

&nbsp;       \*   \*\*网盘授权容错：\*\* 网盘 API 采用 OAuth 验证，Token 会定期失效。在执行备份时检测到 Token 过期/失效，必须立即中断上传，并在软件界面（右上角或居中弹窗）提示：\*\*“网盘授权已过期，请重新绑定”\*\*，并提供重新授权的入口。
