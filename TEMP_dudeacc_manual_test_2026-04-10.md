# DudeAcc CLI 人工测试方案（临时）

## 1. 目标

- 验证开发态 `npm run cli --` 能正常进入交互式 CLI
- 验证交互态可以实际输入字符，不再出现“只显示 `dudeacc>` 但无法输入”的问题
- 验证高频中文命令、上下文记忆、英文原始命令和退出流程

## 2. 测试环境

- 操作系统：Windows 10 / 11
- 终端：PowerShell
- 工作目录：`D:\coding\dude accounting\dude-app`

## 3. 执行前准备

在项目根目录执行：

```powershell
npm install
```

可选预检：

```powershell
npm run typecheck
```

## 4. 测试项

### 4.1 开发态交互入口

执行：

```powershell
npm run cli --
```

预期：

- 先看到构建输出
- 最终进入交互提示符：

```text
dudeacc>
```

- 键盘输入可见，可正常录入字符

### 4.2 帮助命令

在交互界面输入：

```text
help
```

预期：

- 输出 `DudeAcc 交互式 CLI`
- 能看到内建命令列表
- 能看到中文别名列表，如 `登录`、`账套列表`、`选择账套`
- 再次回到 `dudeacc>` 提示符

### 4.3 登录流程（中文补问）

输入：

```text
登录
```

然后按提示输入：

```text
用户名: admin
密码:
```

预期：

- 登录成功后返回 JSON 成功结果
- 输出中出现：
  - `"status": "success"`
  - `"username": "admin"`
- 提示符重新出现

### 4.4 当前用户查询

输入：

```text
我是谁
```

预期：

- 返回当前登录用户信息
- 输出中包含 `admin`

### 4.5 账套列表

输入：

```text
账套列表
```

预期：

- 返回当前账号可访问账套列表
- 若无账套，至少返回成功结构和空列表

### 4.6 选择账套（显式输入）

输入：

```text
选择账套 1
```

预期：

- 输出 `已选择当前账套：1`
- 提示符变成：

```text
dudeacc[ledger:1]>
```

### 4.7 选择账套（补问模式）

输入：

```text
选择账套
```

预期：

- 先打印“可选账套”列表
- 再提示输入账套 ID
- 输入有效账套 ID 后，提示符更新为带 `ledger:<id>` 的形式

### 4.8 选择期间

输入：

```text
选择期间 2026-04
```

预期：

- 输出 `已选择当前期间：2026-04`
- 提示符变成：

```text
dudeacc[ledger:1|period:2026-04]>
```

### 4.9 期间状态（复用上下文）

输入：

```text
期间状态
```

预期：

- 不需要重复输入 `ledgerId`
- 不需要重复输入 `period`
- 返回结果中包含当前期间 `2026-04`

### 4.10 科目余额表（复用上下文）

输入：

```text
科目余额表
```

预期：

- 自动复用当前账套和期间
- 若账套有数据，则返回余额表结果
- 若账套无数据，也应返回结构化成功结果或明确错误，不得卡死

### 4.11 英文原始命令

输入：

```text
ledger list
period status --ledgerId 1 --period 2026-04
```

预期：

- 英文原始命令仍然可用
- 与中文命令行为一致

### 4.12 输出模式切换

输入：

```text
mode json
context
mode pretty
context
```

预期：

- `mode json` 后输出为标准 JSON
- `mode pretty` 后输出回到人类可读模式

### 4.13 上下文清理

输入：

```text
清除期间
上下文
清除账套
上下文
清空上下文
上下文
```

预期：

- `清除期间` 后只移除 `period`
- `清除账套` 后同时清除账套和期间
- `清空上下文` 后上下文完全回到空状态

### 4.14 退出

输入：

```text
exit
```

预期：

- CLI 正常退出
- 返回到 PowerShell 提示符

## 5. 失败判定

出现以下任一情况即判定失败：

- 出现 `dudeacc>` 但键盘无法输入
- 输入字符不回显，且命令无法执行
- 中文命令无响应
- 提示符不更新
- 命令执行后卡住不返回
- `exit` 无法退出
